Wallpaper-viewer
A desktop image gallery application built with Python and Tkinter that lets you browse, manage, and enjoy your wallpaper collection with a clean, modern dark UI.
Features

1.Browse images — Navigate forward/backward through your collection
2.Add photos — Load single or multiple images from anywhere on your system
3.Remove images — Remove any image from the current session
4.Slideshow mode — Auto-advances through all images every 2.5 seconds
5.Image counter — Always shows your current position (e.g. 1 / 8)
6.Filename display — Shows the active image's filename below the viewer
7.Letterbox rendering — All images are centered on a fixed canvas so mixed-size photos never break the layout
