Wallpaper-viewer
A desktop image gallery application built with Python and Tkinter that lets you browse, manage, and enjoy your wallpaper collection with a clean, modern dark UI.
Features

Browse images — Navigate forward/backward through your collection
Add photos — Load single or multiple images from anywhere on your system
Remove images — Remove any image from the current session
Slideshow mode — Auto-advances through all images every 2.5 seconds
Image counter — Always shows your current position (e.g. 2 / 8)
Filename display — Shows the active image's filename below the viewer
Letterbox rendering — All images are centered on a fixed canvas so mixed-size photos never break the layout
